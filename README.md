## ✨ 功能特点

### 核心功能
- ✅ **智能窗口检测** - 只对游戏窗口生效，不影响其他程序
- ✅ **DirectX底层输入** - 使用Windows API，输入更精准
- ✅ **图形化操作界面** - 无需编程知识，配置简单直观
- ✅ **多线程运行** - 界面流畅，脚本运行稳定
- ✅ **配置保存/加载** - 一键保存常用设置，下次直接使用

### 高级特性
- 🔧 **技能循环系统** - 可配置攻击和Buff技能序列
- ⏰ **智能时间控制** - 独立设置攻击间隔和Buff间隔
- 🎯 **窗口焦点检测** - 可选仅当游戏激活时运行
- 📊 **实时状态监控** - 显示攻击次数、运行时间等统计
- 📦 **系统托盘支持** - 支持最小化到后台运行
- 🧪 **按键测试功能** - 测试配置的按键是否有效

## 🖥️ 系统要求

### 最低配置
- **操作系统**: Windows 7 SP1 或更高版本
- **处理器**: Intel Core i3 或同等性能
- **内存**: 2 GB RAM
- **存储空间**: 100 MB 可用空间
- **运行环境**: .NET Framework 4.5+
- **屏幕分辨率**: 1024×768 或更高

### 推荐配置
- **操作系统**: Windows 10/11 (64位)
- **处理器**: Intel Core i5 或更高
- **内存**: 4 GB RAM 或更高
- **存储空间**: 200 MB 可用空间
- **游戏版本**: 冒险岛083版本
